# Mind Blog Frontend

Frontend do projeto **Mind Blog**, desenvolvido em React como parte de um case técnico. 

## Tecnologias 🚀

- React
- TypeScript
- Vite
- React Router DOM
- Tailwind CSS
- shadcn/ui
- Axios
- Lucide React

## Funcionalidades 🖥️

- Home
- Listagem de artigos
- Detalhes do artigo
- Login
- Cadastro de usuários
- Integração com API REST

## Estrutura do Projeto

```text
src/
├── assets/
├── components/
│   ├── layout/
│   └── ui/
├── pages/
├── routes/
├── services/
├── types/
└── main.tsx
```

## Como executar

Clone o repositório:

```bash
git clone <url-do-repositorio>
```

Entre na pasta:

```bash
cd blog-frontend
```

Instale as dependências:

```bash
npm install
```

Execute o projeto:

```bash
npm run dev
```

A aplicação ficará disponível em:

```
http://localhost:5173
```

## Integração

Este projeto consome uma API REST desenvolvida em Node.js e Express para gerenciamento de usuários, autenticação e artigos.
